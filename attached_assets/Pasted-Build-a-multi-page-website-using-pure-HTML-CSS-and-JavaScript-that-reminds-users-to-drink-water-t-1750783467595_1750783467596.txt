Build a multi-page website using pure HTML, CSS, and JavaScript that reminds users to drink water, tracks their daily progress, encourages hydration through gamification, and delivers smart hydration alerts via Push Notifications.

🔑 Core Functional Requirements:
1. 🧍 Personalized Onboarding
Prompt the user to enter their name on first visit.

Store and recall the name using localStorage for a personal UX.

2. 🚰 Smart Hydration Logic
Track hydration goals: 8–9 glasses of water per day.

Use localStorage to manage:

userName

glassesDrankToday

lastDrinkDate

hydrationStreak

totalGlassesDrank

Daily logic:

If user hasn’t drunk water today: show "You haven’t hydrated yet. Drink now!"

If partially hydrated: show progress like "3/8 glasses down. Keep going!"

If goal met: celebrate with "You're fully hydrated! 🥳💧"

3. 🔔 Push Notifications
Request notification permission on app load.

If permission is granted and user hasn’t hydrated yet:

Send push notification:
"Hey [Name], sip some water and slay the day 💧🔥"

Trigger optional vibration with notification on mobile.

Optional: auto-reminder every few hours using setInterval if the tab is open.

4. 🧠 Fun Facts & Hydration Messages
Show rotating fun, poetic, scientific hydration facts like:

“75% of your brain is water. Stay sharp 🧠💧”

“Your mood improves with every sip 💙”

“Hydrate before you dehydrate!”

5. 🌀 3D + Animated Vaporwave UI
Neon glassmorphism styling with glowing, blurred effects.

Spinning 3D elements: cups, droplets, waves.

Animated “Drink” button with bounce/glow.

Water-fill animation when a glass is taken.

6. 📱 Responsive Design
Mobile-first with CSS Flexbox/Grid and media queries.

Buttons scale for mobile and desktop.

Layout adjusts smoothly across all devices.

7. 📊 Gamified Stats & Levels
Show hydration progress X/8 glasses.

Display current streak, total glasses, and levels:

“Thirsty Noob 💧” → “Hydro Hero 🌊” → “Aqua God 🌌”

Visual progress bar and streak badges.

Celebrate streak milestones with animation + haptic feedback.

🗂 Pages Breakdown:
Page	Purpose
index.html	Welcome screen + name prompt
tracker.html	Main hydration logic + fun facts + drink actions
stats.html	Streaks, XP level, total glasses
about.html	App info, reset options, credits (optional)

🔧 Technologies:
HTML5 – For structure and routing

CSS3 – For animations, responsive UI, 3D-style neon effects

JavaScript (Vanilla) – For logic, hydration tracking, push notifications, localStorage, DOM control
